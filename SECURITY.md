# Security Policy

## Supported Versions

Py-Aids can work in python versions above 3.5

| Version | Supported          |
| ------- | ------------------ |
| Python >= 3.5 | :white_check_mark: |
| Python <= 2| :x:                |


## Reporting a Vulnerability

You can report a vulnerability through [Issues](https://github.com/Quarkblue/Py-AIDS/issues)
